About COMPAS

{%endblock page_header%} {% block content %}

COMPAS is a rapid binary population synthesis suite developed by an international team of scientists. It is an updated version of the Single Stellar Evolution (SSE; Hurley, Pols & Tout 2000) and Binary Stellar Evolution (BSE; Hurley, Tout & Pols 2002) written in a modern, object oriented paradigm using the C++ programming language.

The COMPAS code is open source and freely available at https://github.com/TeamCOMPAS/COMPAS

Get in touch at compas-user@googlegroups.com or post an issue at https://github.com/TeamCOMPAS/COMPAS/issues/

See https://compas.science/science.html#papers for a list of papers using COMPAS (or see the ADS )

The results from several studies using COMPAS are publicly available at Zenodo

We are interested to know if you use COMPAS for your research; send us a link to your paper!

This web portal was developed by the Gravitational Wave Data Centre (GWDC) and Astronomy Data and Computing Services (ADACS) through the competitive ADACS Software Support Program.
